## Comparison Table

| Feature | General | Custody | Close Custody |
|---|---|---|---|
| **Title** | خطاب عام | طلب عهدة مالية | طلب إغلاق عهدة مالية |
| **Cost Center row** | Hidden | Shown | Shown |
| **Opening paragraph** | _(none — goes straight to details)_ | "آمل من سعادتكم... صرف عهدة مالية لبرنامج X" | "أرفع لسعادتكم طلب إغلاق عهدة... بعد إتمام الصرف" |
| **Free-text details** | "تفاصيل الخطاب:" + text | "تفاصيل الطلب:" + text | _(none)_ |
| **Amount fields** | _(none)_ | مبلغ العهدة المطلوب + 5K cap note | المبلغ المستخدم + المبلغ المتبقي |
| **Attachments (header)** | Optional (user input) | _(none)_ | Required (user input) |
| **Attachments note** | _(none)_ | _(none)_ | "وسيتم إرفاق المشفوعات الداعمة..." |
| **Closing** | شاكرين لسعادتكم... | شاكرين لسعادتكم... | شاكرين لسعادتكم... |
| **Signature block** | Same for all | Same for all | Same for all |

All three share the same A4 page shell: letterhead background image, Hijri/Gregorian dates in the header overlay, addressee line, greeting, closing, signature block, and page number at the bottom.
