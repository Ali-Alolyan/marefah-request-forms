# مولّد الخطابات - جمعية معرفة (v16)

مولّد خطابات عربي خفيف (Static Web App) يدعم:
- **خطاب عام**
- **طلب مالي عام**
- **طلب عهدة**
- **طلب إغلاق عهدة**
- معاينة **A4** فورية مع كليشة الجمعية
- تصدير **PDF** (متوافق مع Safari/Chrome على Mac و iPhone)

> جميع الخطابات موجهة إلى: **سعادة المدير التنفيذي/ أ.د. محمد عبدالعزيز العواجي حفظه الله**

في **طلب مالي عام** تُعرض تفاصيل الخطاب من إدخال المستخدم، وعند تركها فارغة يظهر موضع نائب: `تفاصيل الخطاب`.

---

## تشغيل محلي (حل مشكلة 404)
**السبب الشائع للـ 404**: تشغيل السيرفر من مجلد غير الذي يحتوي `index.html`.

### الطريقة الأسهل (مُستحسنة)
1) ادخل مجلد المشروع:
```bash
cd letter-tool-v16
```
2) شغّل سيرفر محلي:
```bash
python3 -m http.server 8080
```
3) افتح في المتصفح:
- `http://localhost:8080/` أو `http://localhost:8080/index.html`

### إذا شغّلت السيرفر من المجلد الأب (Parent)
افتح:
- `http://localhost:8080/letter-tool-v16/index.html`

## تصدير PDF (جودة عالية)

- زر **تصدير PDF** يولّد ملف PDF مباشرة **بدون Print/Popups** (رندر Canvas + خط مضمّن)؛ والنتيجة تكون ثابتة على:
  - Safari/Chrome (Mac)
  - Safari/Chrome (iPhone)

### ملاحظات مهمة
- التصدير لا يعتمد على إعدادات الطباعة، لذلك لا يظهر أي جزء من واجهة الموقع داخل PDF.
- تصدير الخطاب الأساسي لا يعتمد على مكتبات خارجية.
- **مرفقات PDF** تستخدم `pdf.js` بتحميل محلي أولاً من:
  - `assets/vendor/pdfjs/pdf.min.mjs`
  - `assets/vendor/pdfjs/pdf.worker.min.mjs`
  وإذا لم تتوفر الملفات محليًا سيتم استخدام CDN كخيار احتياطي.
- على الجوال: إذا كانت الذاكرة منخفضة، قد يخفض التطبيق الدقة تلقائياً لتجنّب التعطّل.

## الجوال (Responsive)
- على الشاشات الصغيرة يظهر شريط تبديل **بالأسفل**: **المدخلات** / **المعاينة**.
- المعاينة تُصغّر تلقائيًا لتناسب عرض الشاشة.

---

## تخصيص مواضع حقول الهيدر
في `css/app.css` ستجد متغيرات (بوحدة mm) لحقول الهيدر:
- `--ov-hijri-top/left`
- `--ov-greg-top/left`
- `--ov-att-top/left`

ولتحريك التاريخين يمين/يسار بسرعة (بالـ px):
- `--ov-date-shift-x` (القيمة الافتراضية: `-5px`)

## استبدال الكليشة (Letterhead)
الصورة المستخدمة للمعاينة:
- `assets/letterhead.png` (خفيفة)

الصورة عالية الدقة للطباعة:
- `assets/letterhead-300.jpg` (تُستخدم تلقائياً عبر srcset)

**مستحسن** أن تكون بدقة 300 DPI (A4 = 2482×3510 px) لضمان PDF نظيف.

بالنسبة للجوال، يوجد ملف متوسط: `assets/letterhead-240.jpg` لتسريع الرندر عند الحاجة.
